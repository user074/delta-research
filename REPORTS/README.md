# REPORTS

Structured evidence from completed runs.

## Contract
- Written by: **Worker**
- Read by: **Supervisor** (to compress into STATE.md)
- Format: `R###.md` — follows `templates/REPORT.template.md`
- Required sections: Result, Signal, Verdict, Confounds, Next tests
- Verdict must be one of: `supports` | `contradicts` | `unclear` | `BLOCKER`
